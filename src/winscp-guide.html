WinSCP 配置说明

1. 安装 WinSCP

确保已安装 WinSCP，默认路径为：D:\工具安装路径\avast杀毒软件工具\WinSCP\WinSCP.com

如果安装在其他位置，请在设置面板中修改 WinSCP 路径。

2. 配置 Session

  1. 打开 WinSCP 图形界面（WinSCP.exe）
  2. 新建站点：协议选 FTP，主机名填 whitelisting.avast.com
  3. 用户名填 ftp_dicloakcom，输入密码
  4. 点击"保存"，站点名称保持默认（即 ftp_dicloakcom@whitelisting.avast.com）
  5. 确认能正常连接后关闭 WinSCP
注意：上传工具使用的是 WinSCP 命令行版本（WinSCP.com），它会自动读取图形界面中保存的 Session。 确保 Session 名称为 ftp_dicloakcom@whitelisting.avast.com。

3. 远程目录

上传目标目录为 /data/，对应 Avast 白名单提交路径。

连接后远程根目录是 /，里面有 data 文件夹。

4. 常见问题

Q: 提示"系统找不到指定的路径"

A: WinSCP.com 路径配置错误，请在设置中检查路径是否正确。

Q: 提示"Host does not exist"

A: Session 名称不对，请在 WinSCP 图形界面中确认保存的站点名称。

Q: 上传卡在 "Host:" 不动

A: WinSCP 没有找到保存的 Session，正在交互式询问。请先在 WinSCP 图形界面中保存站点。
